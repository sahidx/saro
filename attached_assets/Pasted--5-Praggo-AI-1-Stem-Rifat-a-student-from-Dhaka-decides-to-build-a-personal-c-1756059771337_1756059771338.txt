থাকে, যা প্রসেসরকে দ্রুত কাজ করতে সাহায্য করে। সুতরাং, মাদারবোর্ড রিফাতের কেনা যন্ত্রাংশগুলোকে সংযুক্ত করার প্ল্যাটফর্ম হিসেবে কাজ করে। ঘ) উচ্চতর দক্ষতা: রিফাতের গ্রাফিক ডিজাইনের কাজের জন্য শক্তিশালী হয়, তবে শক্তিশালী প্রসেসরও তার পূর্ণ ক্ষমতা ব্যবহার করতে পারে না। রিফাতের ১৬ জিবি র‍্যাম তাকে একই সাথে একাধিক সফটওয়্যার বা বড় ফাইল নিয়ে কাজ করার সুযোগ দেবে, যা প্রসেসরের উপর চাপ কমাবে এবং কম্পিউটারকে ধীর। ফলে শক্তিশালী প্রসেসর থাকা সত্ত্বেও রিফাতের কম্পিউটার ধীর হয়ে যেত। ৩. সমন্বয়ের গুরুত্ব: একটি শক্তিশালী Core i7 প্রসেসরের সাথে যদি মাত্র ৪ জিবি র‍্যাম ব্যবহার করা হয়, তবে তা একটি দুর্বল প্রসেসরের সাথে ৮ জিবি র‍্যাম ব্যবহারের চেয়েও খারাপ পারফরম্যান্স দিতে পারে। কারণ প্রসেসর তথ্যের জন্য অপেক্ষা করতে থাকবে, একে 'বটলনেক' (bottleneck) বলা হয়। বাংলাদেশের প্রেক্ষাপটে একজন ফ্রিল্যান্সারের জন্য সময় অত্যন্ত মূল্যবান। তাই, রিফাতের জন্য একটি শক্তিশালী প্রসেসর এবং পর্যাপ্ত র‍্যামের সমন্বয় নিশ্চিত করা জরুরি। এটি তার কাজের গতি বাড়াবে, উৎপাদনশীলতা বৃদ্ধি করবে এবং ক্লায়েন্টকে সময়মতো কাজ জমা দিতে সাহায্য 3. Stem: A local software company, 'Deshi Coder Ltd.', is developing an inventory management system for a supermarket. When their programmers run the software for testing, it operates very fast, retrieving product data almost instantly. However, if there is a sudden power failure, all the unsaved data entered during the test is lost. To prevent this, they incorporated a feature to permanently save the data on a different type of storage. a. What is RAM? b. Why is RAM considered volatile memory? c. Based on the stem, explain the characteristics of the memory responsible for the initial data loss at 'Deshi Coder Ltd.'. d. Analyze the importance of using both types of memory (the one causing data loss and the one for permanent storage) for the efficient operation of the inventory management system.
বিস্তারিত উত্তর: ক) জ্ঞান: র‍্যাম (RAM) এর পূর্ণরূপ হলো Random Access Memory। এটি কম্পিউটারের একটি অস্থায়ী প্রধান মেমোরি, যেখানে কম্পিউটার চালু থাকাকালীন চলমান প্রোগ্রাম এবং ডেটা সংরক্ষিত থাকে। খ) অনুধাবন: র‍্যামকে উদ্বায়ী বা অস্থায়ী (Volatile) মেমোরি বলা হয় কারণ এটিতে ডেটা সংরক্ষণের জন্য সার্বক্ষণিক বিদ্যুৎ সংযোগের প্রয়োজন হয়। বিদ্যুৎ প্রবাহ বন্ধ হয়ে গেলে র‍্যামে সংরক্ষিত সমস্ত ডেটা বা তথ্য মুছে যায়। একারণেই কম্পিউটার বন্ধ বা রিস্টার্ট করলে র‍্যামের সব তথ্য হারিয়ে যায়। এটি এর গঠনগত কারণেই অস্থায়ী। গ) প্রয়োগ: উদ্দীপকে 'দেশি কোডার লিমিটেড' এর সফটওয়্যার পরীক্ষার সময় বিদ্যুৎ চলে যাওয়ায় যে ডেটা হারিয়ে গিয়েছিল, তার জন্য দায়ী মেমোরিটি হলো র‍্যাম (RAM)। এটি কম্পিউটারের প্রধান মেমোরি। এর বৈশিষ্ট্যগুলো হলো: ১. অস্থায়ী প্রকৃতি: র‍্যাম একটি উদ্বায়ী মেমোরি। বিদ্যুৎ সংযোগ বিচ্ছিন্ন হওয়ার সাথে সাথে এর মধ্যে থাকা সকল তথ্য মুছে যায়, যা উদ্দীপকের ঘটনায় ঘটেছে। ২. উচ্চ গতি: উদ্দীপকে বলা হয়েছে সফটওয়্যারটি খুব দ্রুত কাজ করছিল এবং পণ্যের ডেটা তাৎক্ষণিকভাবে পাচ্ছিল। এর কারণ হলো প্রসেসর সরাসরি র‍্যাম থেকে ডেটা আদান-প্রদান করে, যা হার্ডডিস্ক বা এসএসডি-র চেয়ে অনেক বেশি গতিসম্পন্ন। ৩. সীমিত ধারণক্ষমতা: সাধারণত র‍্যামের ধারণক্ষমতা হার্ডডিস্কের চেয়ে অনেক কম থাকে, তবে এটি খুব দ্রুত কাজ করার জন্য তৈরি। সফটওয়্যারটি চলার সময় প্রয়োজনীয় ডেটাগুলোই শুধু র‍্যামে লোড হয়। মূলত, সফটওয়্যারটি চালু অবস্থায় সকল ইনপুট করা ডেটা র‍্যামে অবস্থান করছিল এবং সেভ না করার কারণে বিদ্যুৎ চলে যাওয়ায় সেই ডেটাগুলো হারিয়ে যায়। ঘ) উচ্চতর দক্ষতা: ইনভেন্টরি ম্যানেজমেন্ট সিস্টেমের মতো একটি সফটওয়্যারের কার্যকর পরিচালনার জন্য র‍্যাম (অস্থায়ী মেমোরি) এবং হার্ডডিস্ক/এসএসডি (স্থায়ী মেমোরি) উভয়ই অপরিহার্য। নিচে
